# 🎻 CM Šafrán & CM Valášek Zlín

Oficiální webové stránky cimbálových muzik ze Zlína. Tento projekt slouží jako digitální vizitka pro prezentaci obou těles, jejich členů, historie a plánovaných akcí.

## 🚀 O projektu
Web je postaven jako moderní, plně responzivní jednostránková prezentace (SPA) s dedikovanými podstránkami pro významné události.

### Klíčové funkce:
- **Responzivní design:** Optimalizováno pro mobily, tablety i PC pomocí Bootstrapu 5.
- **Interaktivní galerie:** Využití Lightbox2 pro prohlížení fotografií z vystoupení.
- **Kalendář akcí:** Přehledný seznam plánovaných "hraček" a koncertů.
- **USA Tour 2025:** Speciální sekce věnovaná americkému turné s reportáží a fotodokumentací.

## 📁 Struktura složek
- `/` - Hlavní stránky (`index.html` pro Šafrán, `valasek.html`)
- `/styles` - CSS styly (hlavní designový manuál)
- `/images` - Strukturovaná úložiště fotografií
  - `/safran` - Fotky CM Šafrán
  - `/valasek` - Fotky CM Valášek
  - `/extras/loga` - Logotypy a ikonky (favicons)
- `/amerika` - Podstránka a data k turné v USA

## 🛠️ Technologie
- **HTML5** & **CSS3** (Custom design s využitím moderních filtrů a blur efektů)
- **Bootstrap 5** (Layout a mřížka)
- **Boxicons** (Ikonografie)
- **Google Fonts** (Písma Alegreya a Open Sans)

## ✍️ Správa webu
Web je hostován na **GitHub Pages**. Pro aktualizaci akcí nebo přidání fotek stačí editovat příslušné sekce v HTML kódu a nahrát změny do repozitáře.
